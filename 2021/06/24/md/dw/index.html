# links 链接块

以下正文

本功能参考 NexT，基于 Hexo Tag 功能，用来建立友链或其他网址链接功能。
文章中的使用格式：

  1. 使用 links 标签块，包围 yml 语法书写的内容，字段包括
site站点名称必填
owner管理员名字可选，默认为 site 的值
url站点链接必填
desc站点描述可选，默认为 url 的值
image站点图片可选，默认为 images/404.png
color方块颜色可选，默认为 #666

<!-- :- 表示 right -: left :---: 居中 -->

{% links %}
- site: #站点名称
  owner: #管理员名字
  url: #站点网址
  desc: #简短描述
  image: #一张图片
  color: #颜色代码
{% endlinks %}

举个栗子：

優萌初華

琉璃的医学 & 编程笔记

優萌初華

https://shoka.lostyu.me

優萌初華

琉璃的医学 & 编程笔记

  1. 以上标签块里的内容，还可以保存到一个 yml 文件中，然后使用 linksfile 标签。
    其中， path 位于 <root>/source 目录下。

{% linksfile friends/_data.yml %}
friends 是 source 文件夹下面的一级目录
